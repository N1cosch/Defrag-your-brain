
# Defrag your brain

Retro Wohlfühl App, die eine Defragmentierung **nur simuliert**.  
Keine echten Daten, keine Optimierung, nur entspannte Pixel Ordnung.

## Features
- Disk Größe: klein / groß / riesig
- Speed Regler + Turbo
- Fortschritt + Fragmentierung + Moves
- Meditative Status Sprüche
- Optionaler HDD Sound (WebAudio)

## Usage
Seite öffnen → Disk wählen → **Start** → zuschauen.  
Sound optional per Button.

## Hinweis
Das ist reine Simulation. Kein Tool.
